    Wishlist

    We all like to dream, but sometimes it's not enough.

    These are the things I'd love to have in my life but resist to come to my real world.

    If you have any lead, as smallest as it can be, please contact me.

    Self hosted search engine¶

    It would be awesome to be able to self host a personal search engine that performs priorized queries in the data sources that I choose.

    This idea comes from me getting tired of:

    • Forgetting to search in my gathered knowledge before going to the internet.
    • Not being able to priorize

    That indexes:

    • My blue and red books.
    • Awesome lists
    • My browsing history
    • Blogs (dwern, drymer)
    • learn-anything.org
    • musicbrainz
    • themoviedb
    • reddit
    • stackoverflow

    Each source could be added as a plugin to let people develop their own easily.

    I'd also like to be able to rate in my browsing the webpages so they get more relevance in future searches. That rating can also influence the order of the different sources.

    It could monitor too what links I follow after a search in the frontend.

    It will archive the rated websites to avoid linkrot.

    A possible architecture could be:

    • A flask + Reactjs frontend.
    • An elasticsearch instance to index the data
    • A neo4j or knowledge graph to get relationships

    If we use a knowledge graph, we could federate to ask other nodes and help discover or priorize content.

    The browsing could be related with knowledge graph tags

    We can also have integration with Anki after a search is done
